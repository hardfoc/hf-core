Detailed Description

CRTP adapter that turns a BaseUart& into a transport the alicat_basis2::Driver can drive.

Notes
  • BaseUart::Read returns an error code, not a byte count, so we trust the contract that the underlying driver fulfills the request completely or returns a non-success error (timeout / no data).
  • delay_ms_impl uses vTaskDelay so the driver can satisfy the Modbus-RTU 3.5-character idle gap when needed.
