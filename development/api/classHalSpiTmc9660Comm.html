Detailed Description

CRTP SPI communication adapter bridging BaseSpi/BaseGpio to tmc9660::SpiCommInterface.

Handles TMCL/bootloader SPI transfers, four host-side control pins (RST, DRV_EN, FAULTN, WAKE) with configurable active-level polarity, and platform-aware delays.

Note
Does NOT own the BaseSpi or BaseGpio instances.
