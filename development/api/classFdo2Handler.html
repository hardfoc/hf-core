Detailed Description

HAL handler wrapping one FDO2-G2 oxygen probe on a BaseUart.

Ownership: the handler holds a reference to the supplied BaseUart (the HAL owns the actual EspUart via CommChannelsManager); the handler owns the CRTP adapter + the templated driver instance.
