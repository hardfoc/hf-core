Detailed Description

Unified handler for TMC9660 motor controller with GPIO, ADC, and temperature integration.

Provides HAL-level integration for a TMC9660 motor driver using BaseSpi/BaseUart and BaseGpio. Architecture: CRTP comm adapters → typed TMC9660 driver → non-templated handler facade. Inner classes (Gpio, Adc, Temperature) adapt TMC9660 peripherals to HardFOC base interfaces.
