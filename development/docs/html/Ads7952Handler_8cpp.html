Detailed Description

Implementation of the ADS7952 12-channel SAR ADC handler.

Implements the CRTP SPI bridge, BaseAdc interface, lazy initialization, and thread-safe operations for the ADS7952 ADC. Follows the same architecture as As5047uHandler.cpp.
